# OptoLearn: Advanced Optoelectronics Learning Module

An ultra-premium, interactive educational dashboard built to make learning physics engaging and visually stunning. Moving away from traditional static textbook models, this application provides an immersive, experimental environment for students to learn about optoelectronic devices.

## Features

* **Real-time Simulations:** Interactive Light-Dependent Resistor (LDR) environments allowing users to adjust light levels and see real-time resistance outputs.
* **Animated Device Physics:** Intuitive, animated deep-dives into the inner workings of photodiodes and phototransistors.
* **Ultra-Premium UI/UX:** Designed with a "Hardware Dashboard" aesthetic, utilizing edge-lit glassmorphism, 3D parallax tilt cards, magnetic buttons, and animated mesh backgrounds.
* **Gamified Evaluation:** A state-driven interactive quiz system designed to test knowledge retention in a rewarding, engaging way.

## Technologies Used

* **HTML5:** Semantic structure and layout.
* **Vanilla CSS / CSS3:** Custom animations, mesh backgrounds, glassmorphism, edge-lighting, and responsive design. 
* **Vanilla JavaScript (ES6+):** DOM manipulation, state management for the interactive quiz, and parallax/3D calculations for UI elements.

## Getting Started

Since this is a vanilla HTML/CSS/JS project, you don't need to install any heavy dependencies or build tools to run it.

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/opto-learn.git
   ```
2. **Navigate to the directory:**
   ```bash
   cd opto-learn
   ```
3. **Open the project:**
   Simply double click the `index.html` file to open it in your default web browser, or use a local development server like VS Code's Live Server extension for a better experience.
